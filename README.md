# Agenda Telefônica - Frontend

Aplicação web de agenda telefônica com CRUD de contatos, desenvolvido com node.js e React.

## Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS

## Funcionalidades

- Adicionar Contato
- Listar Contatos
- Deletar Contato

## Em desenvolvimento

- Login de Usuários
- Cadastro de Usuários

## Conceitos Aplicados

- Componentização
- Hooks (useState)
- Separação de responsabilidade
